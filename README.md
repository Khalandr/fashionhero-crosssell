# FashionHero — Dobierz do tego (Side Quest: Road Not Taken)

Second prototype for AIPH2 Q1.3 / Side Quest "Road Not Taken". An interactive mock of a FashionHero product page with a **Dobierz do tego** cross-sell carousel embedded.

Different hypothesis than the first prototype ([fashionhero-plus](https://github.com/Khalandr/fashionhero-plus)): instead of testing whether buyers will pay for a Premium membership, this one tests whether algorithmic outfit-completion cross-sell can lift basket size on existing traffic — no behaviour change required from buyers.

**Live:** https://khalandr.github.io/fashionhero-crosssell/

Sister prototype: FashionHero Plus (buyer subscription) — https://khalandr.github.io/fashionhero-plus/
